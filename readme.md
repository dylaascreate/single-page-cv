# Single-Page CV
[roadmap.sh
](https://roadmap.sh/projects/single-page-cv)

A simple, responsive single-page CV (Curriculum Vitae) template built with HTML and CSS. Perfect for showcasing your skills, experience, and education online or as a printable PDF.

## Features

- Clean, minimalist design
- Fully responsive layout (works on desktop and mobile)
- Easy to customize your personal information, experience, skills, and links
- Quick export to PDF (use browser's 'Print' > 'Save as PDF')
- Ready for GitHub Pages hosting

## Getting Started

1. **Clone or Download** this repository to your computer.
2. **Open** `index.html` in your favorite code editor.
3. **Replace** the placeholder sections (`Name`, `Profile`, `Skills`, `Experience`, etc.) with your own information.
4. **Customize** colors, fonts, sections, or layout by editing the styles in `style.css`.
5. **Preview** your CV by opening `index.html` in a web browser.
6. **Publish** by pushing your finished CV to GitHub and enabling GitHub Pages.

## Customization Tips

- Modify `index.html` to add or remove sections (skills, projects, contact, etc.)
- Change style by updating CSS in `style.css`
- Add links to your GitHub, LinkedIn, email, or other profiles
